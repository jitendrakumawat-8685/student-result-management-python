# Student Result Management System
A Python project to manage student results and grades.

## Features
- Add student with marks
- Calculate grade automaticaly
- Show individual result
- View top performance
- Save results to file (no duplication)

## Technologies Used
- Core Python only
- File Handling
- Function & Dictionaries

## How to Run
1. Open Jupter Notebook
2. Open the .ipynb file
3. Run all cells
4. Use the menu to add students

## Concepts Used
 - Functions
 - Loops
 - Dictionaries
 - File Handling
 - Exception Handling
